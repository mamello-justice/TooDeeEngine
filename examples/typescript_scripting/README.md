# TypeScript Example

Movement and collision detection using TypeScript scripting.

## Overview

This example demonstrates how to use TypeScript for scripting in TooDeeEngine. It covers:
- Initializing the GameEngine with TypeScript support

## Running the Example

```bash
just example TypesScriptScripting
```

## What You'll Learn

- How to set up TypeScript scripting in TooDeeEngine
- Basic movement and collision detection using TypeScript scripts

## Source Files

- `TypesScriptScripting.hpp` - Header file with game setup
- `TypesScriptScripting.cpp` - Implementation file
- `script.ts` - TypeScript script for game logic
