# GridExample

Demonstrates grid-based game mechanics and pathfinding concepts using TooDeeEngine.

## Overview

This example showcases:
- Grid-based movement and collision
- Tile-based game structure
- Pathfinding concepts
- Grid rendering and management

## Running the Example

```bash
just example GridExample
```

## What You'll Learn

- Setting up a grid-based game world
- Entity movement within a grid system
- Grid-based collision detection
- Rendering tile-based environments

## Source Files

- `GridExample.hpp` - Header file with grid system
- `GridExample.cpp` - Implementation file

## Features

- Tile-based level system
- Grid-aligned entity movement
- Efficient grid queries for collision detection
