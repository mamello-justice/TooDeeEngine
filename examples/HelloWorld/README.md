# HelloWorld Example

A minimal example demonstrating basic TooDeeEngine setup and sprite rendering.

## Overview

This is the simplest example to get started with TooDeeEngine. It shows:
- Engine initialization
- Creating a basic game window
- Rendering simple sprites

## Running the Example

```bash
just example HelloWorld
```

## What You'll Learn

- How to initialize the GameEngine
- Basic entity creation and component management
- Simple sprite rendering
- Main game loop structure

## Source Files

- `HelloWorld.hpp` - Header file with game setup
- `HelloWorld.cpp` - Implementation file
