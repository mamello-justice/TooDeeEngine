# JavaScript Example

Movement and collision detection using JavaScript scripting.

## Overview

This example demonstrates how to use JavaScript scripting for scripting in TooDeeEngine. It covers:
- Initializing the GameEngine with JavaScript scripting support

## Running the Example

```bash
just example native_scripting
```

## What You'll Learn

- How to set up JavaScript scripting in TooDeeEngine
- Basic movement and collision detection using JavaScript scripts

## Source Files

- `JavascriptScripting.hpp` - Header file with game setup
- `JavascriptScripting.cpp` - Implementation file
- `bounding_box_collision_script.js` - JavaScript script for bounding rectangles to screen
- `bounding_circle_collision_script.js` - JavaScript script for bounding circles to screen
