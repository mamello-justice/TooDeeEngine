# MovingShapes Example

A more complex example demonstrating physics-based movement and collision detection.

## Overview

This example builds on the basics of TooDeeEngine to show:
- Physics-based movement of shapes
- Collision detection and response
- More advanced entity and component management
- Using the physics system for realistic interactions

## Running the Example

```bash
just example moving_shapes
```

## What You'll Learn

- How to set up physics components for entities
- Implementing movement based on physics forces
- Collision detection and response between shapes
  
## Source Files

- `MovingShapes.hpp` - Header file with game setup
- `MovingShapes.cpp` - Implementation file
