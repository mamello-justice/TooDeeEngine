# Lua Scripting Example

Movement and collision detection using Lua scripting.

## Overview

This example demonstrates how to use Lua scripting for scripting in TooDeeEngine. It covers:
- Initializing the GameEngine with Lua scripting support

## Running the Example

```bash
just example native_scripting
```

## What You'll Learn

- How to set up Lua scripting in TooDeeEngine
- Basic movement and collision detection using Lua scripts

## Source Files

- `NativeScripting.hpp` - Header file with game setup
- `NativeScripting.cpp` - Implementation file
- `CollisionBoxScript.cpp` - Lua script for binding collision box entities within screen
- `CollisionCircleScript.cpp` - Lua script for binding collision circle entities within screen
