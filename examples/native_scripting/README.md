# NativeScripting Example

Movement and collision detection using Native/C++ scripting.

## Overview

This example demonstrates how to use Native/C++ scripting for scripting in TooDeeEngine. It covers:
- Initializing the GameEngine with Native/C++ scripting support

## Running the Example

```bash
just example native_scripting
```

## What You'll Learn

- How to set up Native/C++ scripting in TooDeeEngine
- Basic movement and collision detection using Native/C++ scripts

## Source Files

- `NativeScripting.hpp` - Header file with game setup
- `NativeScripting.cpp` - Implementation file (Includes onBoundingBoxCollision and onBoundingCircleCollision script functions)
